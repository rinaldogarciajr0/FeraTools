# FeraTools

Plugin unico para QGIS que reune tres ferramentas:

- criar ou atualizar campo de area em hectares
- exportar camada vetorial para Shapefile
- criar ou atualizar campos de coordenadas X e Y

## Estrutura

Use apenas a pasta `FeraTools` como plugin instalavel no QGIS. No QGIS, ele aparecera como `FeraTools`.

Os diretorios originais foram mantidos fora dela apenas como referencia:

- `area_ha_plugin`
- `exportar_shp_contexto`
- `pegar_coordenadas`

## Instalacao

1. Compacte a pasta `FeraTools` em `.zip`
2. No QGIS, use `Plugins > Gerenciar e Instalar Plugins... > Instalar a partir do ZIP`
